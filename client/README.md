# VORLAGE

VORLAGE is an ultra-premium, zero-latency, voice-controlled architectural canvas that translates speech into spatial node-graphs via a continuous WebSocket stream.

## High-Performance Architecture

Built for modern AI agencies and high-end tech teams, VORLAGE utilizes:
- **Frontend Engine**: React, Vite, Framer Motion for cinematic 60fps animations, and Tailwind CSS.
- **Node Graphing**: React Flow integrated with Zustand for dynamic state management.
- **Backend Service (Python SDK)**: FastAPI streaming endpoints integrating Deepgram (STT) and OpenAI (GPT-4o) for instantaneous spatial intent parsing.

## 🚀 Quick Start

### 1. Install Dependencies
```bash
npm install
```

### 2. Run the Development Server
```bash
npm run dev
```

### 3. Build for Production
```bash
npm run build
```

## Features
- **Cinematic UI/UX**: Glassmorphism, dynamic islands, magnetic cursors, and custom audio waveforms.
- **Componentized Structure**: Scalable React architecture organized into Layouts, Sections, and UI elements.
- **Built-in Authentication Pages**: Seamless routing to Login, Early Access (Sign Up), and Forgot Password screens.
- **Zero-Latency Mockups**: Demonstrated capabilities capable of processing 10k+ req/sec with <50MB footprint.

## Directory Structure
- `/src/pages`: Application views (`Home`, `Login`, `SignUp`).
- `/src/components/layout`: Global structures like `Header` and `Footer`.
- `/src/components/sections`: Landing page blocks (`Hero`, `Features`, `LiveMockup`).
- `/src/components/ui`: Reusable primitive components.
